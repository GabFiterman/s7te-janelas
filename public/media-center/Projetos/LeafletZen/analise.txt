# Leaflet-Zen: Painel de Controle e Arquitetura de Mapas

O Leaflet-Zen é um projeto de simulação de painel de controle geográfico, diretamente derivado da minha experiência recente em meteorologia. O foco aqui foi na qualidade da arquitetura e na gestão de estados complexos em tempo real.

Este projeto demonstra meu domínio sobre um stack robusto: React, TypeScript e Redux, juntamente com Tailwind CSS para estilização eficiente.

## Pontos Chave para Contratação:
- **Arquitetura Sólida** (_Redux & TS_): O projeto utiliza Redux para gerenciar estados globais e fluxos de dados complexos (como a configuração inicial do mapa, pontos de interesse, áreas e perímetros) de forma previsível e type-safe.

- **Design Modular**: O código foi minuciosamente refatorado seguindo o Design Pattern Atomic Design e faz uso intensivo de Hooks Personalizados. Essa modularidade garante escalabilidade e fácil manutenção do painel.

- **Gestão Completa de Features**: O desenvolvimento foi documentado via Pull Requests detalhados, que cobrem desde a integração da API simulada (`json-server`) até a implementação de funcionalidades avançadas de mapa, como Leaflet Draw para desenho dinâmico de áreas e perímetros.

O Leaflet-Zen comprova minha capacidade de transformar requisitos complexos em um produto funcional, estável e bem arquitetado, seguindo as melhores práticas de desenvolvimento, organização (_Kanban/SCRUM_), e qualidade de código (_ESLint/Prettier_).
